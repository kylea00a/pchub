## PCHUB Host (Windows)

Planned responsibilities:
- Desktop capture (DXGI Desktop Duplication)
- Hardware encode (NVENC/AMF/QSV)
- WebRTC send video/audio
- Receive input (mouse/keyboard/controller) over data channel and inject locally

Install target:
- `C:\Program Files\PCHUB\Host\`
- Service: `PCHUBHostService`
- Persistent state/config: `C:\ProgramData\PCHUB\`

This folder is scaffold-only for now.
